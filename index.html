Attry-Me

Attry-Me uses our specified custom attributes, which the user will define on their form elements. The function pulls these custom attributes and uses their values to validate the form, element by element, and returns true/false. You include the file at the top of your page, and you just add the custom elements to your form, and plop the function where you want it to validate (e.g. on the form submit, or submit button click).

Example of a form element with the Attry-Me custom attributes:

Name: <input type='text' id='attry-me-sample-text' value='Enter your Name' length='30' e_required='true' e_default='Enter your Name' e_type='string'>

Age: <input type='text' id='attry-me-sample-number' value='0' length='10' e_required='false' e_default='0' e_type='rangebetween' e_max='99' e_min='18'>

Then you simple add the function and pass in the form name via the submit button or the form element:

<form name='attry-me-sample-form' id='attry-me-sample-form' action'/sample-file.html' method='post' onsubmit='return AttryMe("attry-me-sample-form");'>

<input type='submit' id='attry-me-submit-btn' value='Submit' onclick='return AttryMe("attry-me-sample-form");'>

The code does the rest!
